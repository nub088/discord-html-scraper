Want to scrape all the data from one particular user on discord who has been posting for several years? This is the python script for you. all you have to do is find the user's unique identifier in the HTML file.

Extract the discord channel using https://github.com/Tyrrrz/DiscordChatExporter.  

then run open the html file, locate the user's unique ID.  

copy/paste it into TARGET_ID.
